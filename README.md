# DistributeX Worker Agent

CLI tool for contributing computing resources to the DistributeX network.

## Installation

\`\`\`bash
npm install -g @distributex/worker
\`\`\`

## Usage

\`\`\`bash
distributex-worker --api-key YOUR_API_KEY
\`\`\`

Get your API key from: https://your-site.pages.dev/dashboard

## Quick Install (Linux/macOS)

\`\`\`bash
curl -sSL https://your-site.pages.dev/install.sh | bash
\`\`\`
